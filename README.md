# RandomBodyBuilding-web

랜덤보디빌딩 웹UI

## 요구사항

- 버튼을 눌렀을 때 아무 응답이 없으면 안 됩니다!!
- 기존 풀잇글, 다시말해 comment는 함부로 지우지 마십시오.

## 코드 스타일

- 파일마다 시작 부분에 그 파일의 구실을 밝히는 풀잇글을 다십시오.
- 직계존속 꾸러미나 직계비속 꾸러미를 임포트하는 경우에는 상대 경로 임포트를 쓰십시오. 나머지 경우(직계 형제·자매 꾸러미 포함)는 절대 경로 임포트를 쓰십시오.
- 순환 임포트는 쓰지 마십시오.
- 지연 임포트는 쓰지 마십시오.
- 임포트 폴백은 쓰지 마십시오.
- `const` 키워드와 `let` 키워드는 쓰지 마십시오. 대신 `var` 키워드를 쓰십시오.
- `class` 문법은 쓰지 마십시오. 대신 프로토타입을 쓰십시오.
- 다음 낱말은 JS 공동체 관례를 따르는 대신 전체를 대문자로 작성하십시오:
   - `HTML`
   - `JS`
   - `CSS`

## 배포 아키텍처

클라우드플레어 페이지를 써서 `public` 폴더를 호스팅합니다.
